# Code of Conduct

## Our Standard

This project expects respectful, constructive, and technically grounded collaboration.

Examples of behavior that contribute to a positive environment:

- giving clear and actionable feedback
- assuming good intent while still being rigorous
- welcoming contributors with different levels of experience
- focusing criticism on code, decisions, and evidence rather than people

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- deliberate intimidation or exclusion
- publishing private information without permission
- repeatedly derailing technical discussion in bad faith

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, wiki edits, and other contributions that violate this code of conduct.

## Scope

This code of conduct applies in project spaces such as issues, pull requests, discussions, and review threads.
